# multiscale-mapping
Optimal method for multiscale analysis of scattered observations

Use map_data.m to run mapping; see this script for explanation of the various steps and associated functions

For a complete explanation of this method, see: Gray, A. R. and S. C. Riser, 2015. A method for multiscale optimal analysis with application to Argo data. J. Geophys. Res. Oceans, 120 (6), 4340-4356.
